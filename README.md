# MSDOS-like link shortener

![An example of how a web application works](./public/interface.png)

# About 
This is client-side of [**MSDOS link shortener**]() app. This app is designed to make **links** easy to read and paste it into other sites.

# Client-side Technology Stack 
1. JS (ES6) 
2. React (18.2.0)
3. Redux (4.2.1)
4. React-bootstrap (2.7.0) (*using Bootstrap grid*)
5. Axios (1.3.2) (*REST API*)
